# AWS Service Quotas

- `AWS Service Quotas`                      AWSサービスのクォータ（制限値）を一元管理・表示・引き上げリクエストできるサービス

## Overview
- クォータ管理:                             AWS サービスの==使用上限==を一元管理 / 上限==引き上げリクエスト==も可能

---

## 主な機能

| 機能 | 説明 |
|------|------|
| **クォータ確認** | 各サービスの上限値を一覧表示 |
| **引き上げリクエスト** | コンソールから上限緩和を申請 |
| **CloudWatch 連携** | クォータ使用率の==アラーム==設定 |

---

## SAA 出題パターン

| シナリオ | 回答 |
|---------|------|
| 「サービスの==使用上限==を確認・管理したい」 | ==Service Quotas== |
| 「==クォータ引き上げ==をリクエスト」 | ==Service Quotas== |
| 「クォータ使用率を==監視・アラート==」 | ==Service Quotas + CloudWatch== |
