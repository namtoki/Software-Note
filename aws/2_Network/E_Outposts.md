# AWS Outposts

## Overview
- オンプレ AWS:   ==!高額== / AWS が顧客のデータセンターや施設に==設置・保守・管理== / ==シングルテナント== / ==Region の接続必須==

## ユースケース

| シナリオ | Outposts が適切な理由 |
|---------|---------------------|
| データレジデンシー | 法規制でデータをオンプレミスに保持 (HIPAA, GDPR等) |
| 低レイテンシー要件 | ローカル処理が必要なワークロード |
| ハイブリッドクラウド | オンプレと AWS を同じ API で統合管理 |
