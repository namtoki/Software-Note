# Amazon EventBridge

```
イベントソース                     ターゲット
┌─────────────┐                  ┌──────────────┐
│ AWS サービス│                  │   Lambda     │
│ (S3, EC2等) │                  │   SQS/SNS    │
├─────────────┤  → EventBridge → │Step Functions│
│    SaaS     │     (ルール)     │   Kinesis    │
│ (Zendesk等) │                  │   API GW     │
├─────────────┤                  │     etc.     │
│ カスタムApp │                  └──────────────┘
└─────────────┘
```

## Associate Level
- イベントバス:               ==イベント駆動==アーキテクチャ / AWS, SaaS, カスタムアプリからのイベントをルーティング
  - 主な機能:
    | 機能 | 説明 |
    |------|------|
    | ==イベントバス== | イベントを受信・ルーティング |
    | ==ルール== | イベントパターンでフィルタリング → ターゲットに送信 |
    | ==スケジュール== | cron/rate 式で定期実行 |
