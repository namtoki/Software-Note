# AWS Wavelength

- `AWS Wavelength`                          ==!Region より 20〜30% 高い== / 通信事業者の5Gネットワーク内に配置 / 5Gデバイスからネットを経由せずに直接AWSにアクセス

## Overview

- **定義**: ==通信事業者==のデータセンター内に AWS インフラを配置
- **特徴**:
  - ==5G エッジ==での超低レイテンシー
  - 5G デバイスから==インターネットを経由せず==直接アクセス
  - ==マルチテナント==

---

## ユースケース

| シナリオ | Wavelength が適切な理由 |
|---------|------------------------|
| ==5G モバイルアプリ== | 5G ネットワーク内で処理 |
| AR/VR | 超低レイテンシーが必須 |
| 自動運転車 | リアルタイム処理 |

---

## Edge サービスの選択基準 (SAA 頻出)

| シナリオ | 正解 |
|---------|------|
| データを==物理的にオンプレミスに保持==したい | ==Outposts== |
| ==AWS 管理==インフラで都市圏の低レイテンシーが必要 | ==Local Zones== |
| ==5G== デバイス向けの超低レイテンシー | ==Wavelength== |

---

## SAA 出題パターン

1. 「==5G== モバイルユーザー向けの超低レイテンシーアプリ」
   → ==Wavelength==

2. 「モバイルデバイスからインターネットを経由せずに AWS にアクセスしたい」
   → ==Wavelength==
