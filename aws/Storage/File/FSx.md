# Amazon FSx

## Overview
- フルマネージドファイルシステム:     4種類のファイルシステムを提供 / 用途: 既存ワークロードの移行、高性能ファイル共有
  - `FSx for Windows File Server`:      ==Windows共有 / SMB / AD統合== / Multi-AZ 対応
  - `FSx for Lustre`:                   ==HPC、機械学習、動画処理== / ==S3をデータリポジトリとして連携==
  - `FSx for NetApp ONTAP`:             マルチプロトコル (NFS+SMB+iSCSI)
  - `FSx for OpenZFS`:                  Linux移行 / NFS
