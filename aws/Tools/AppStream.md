# Amazon AppStream 2.0

## Overview
- アプリストリーミング:                   ==Windows アプリ==を Web ブラウザ経由で配信 / デスクトップ全体ではなく==アプリ単位==

---

## AppStream vs WorkSpaces (SAA 頻出)

| 項目 | ==AppStream 2.0== | ==WorkSpaces== |
|------|------------------|----------------|
| **配信対象** | ==アプリケーション== | ==デスクトップ全体== |
| **用途** | 特定アプリの配信 | 仮想デスクトップ (VDI) |
| **OS** | Windows | Windows / Linux |
| **永続性** | ==非永続== (セッション終了で消去) | ==永続== (データ保持) |
| **課金** | 使用時間 | 月額 or 時間 |

---

## SAA 出題パターン

| シナリオ | 回答 |
|---------|------|
| 「==特定アプリ==をブラウザで配信したい」 | ==AppStream 2.0== |
| 「==仮想デスクトップ== (VDI) が必要」 | ==WorkSpaces== |
| 「アプリを==非永続==でストリーミング」 | ==AppStream 2.0== |
| 「ユーザーに==フルデスクトップ==を提供」 | ==WorkSpaces== |
