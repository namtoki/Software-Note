# AWS Device Farm

AWS が提供する**アプリケーションテストサービス**。

## 仕組み

AWS のデータセンター（主に米国）に**物理的なスマートフォンやタブレットが大量にラックに設置**されている。

```
+----------------------------------+
|  AWS Device Farm 施設            |
|  +------+ +------+ +------+      |
|  | iPhone| |Galaxy| |Pixel | ... |
|  +------+ +------+ +------+      |
|      |       |        |          |
|      +-------+--------+          |
|              |                   |
|      ネットワーク接続             |
+----------------------------------+
               |
          インターネット
               |
         あなたの PC
```

- **エミュレータではなく実機** - 実際の端末なので、カメラ、GPS、センサーなども動作する
- **常時メンテナンス** - AWS が端末の管理・更新を担当
- **隔離環境** - テスト後はデータがクリーンアップされる

## 主な機能

1. **実機デバイスでのテスト** - 物理的な Android・iOS デバイス上でモバイルアプリやウェブアプリをテストできる

2. **自動テスト** - 複数のデバイスで同時に自動テストを実行し、スクリーンショット、ログ、パフォーマンスデータを収集

3. **リモートアクセス** - 実機デバイスにリモートでアクセスし、手動で操作・デバッグが可能

4. **デスクトップブラウザテスト** - Selenium を使用したデスクトップブラウザでのウェブアプリテスト

## ユースケース

- モバイルアプリの互換性テスト（様々な機種・OS バージョン）
- UI/UX の動作確認
- パフォーマンステスト
- 本番環境に近い条件でのデバッグ

## 料金体系

- **従量課金**: デバイス使用時間（分単位）で課金
- **定額プラン**: 月額固定で無制限使用
