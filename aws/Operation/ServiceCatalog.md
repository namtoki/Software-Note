# AWS Service Catalog

==承認済みの IT サービス==を一元管理・展開するサービス。

## 特徴
- `ポートフォリオ`:                    製品のコレクション / アクセス制御単位
- `製品`:                              ==CloudFormation テンプレート==をパッケージ化
- `制約`:                              起動制約、通知制約、テンプレート制約
- `バージョン管理`:                    製品のバージョンを管理

## ユースケース
- 承認済みの構成のみをユーザーに提供
- セルフサービスでの環境構築 (ガバナンス維持)
- 組織全体で標準化されたリソースを展開

## SAA試験ポイント
- ==承認済みサービスの一元管理== → Service Catalog
- ==CloudFormation ベース== → 製品はテンプレートで定義
- ==ガバナンス + セルフサービス== → Service Catalog
