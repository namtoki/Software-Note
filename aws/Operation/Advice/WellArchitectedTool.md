# AWS Well-Architected Tool

## Overview
- アーキテクチャレビュー:                   ==6 つの柱==に基づきワークロードを評価 / 改善点を特定して==レポート==を生成
  - 6 つの柱:
    | 柱 | 説明 |
    |----|------|
    | **Operational Excellence** | 運用の優秀性 |
    | **Security** | セキュリティ |
    | **Reliability** | 信頼性 |
    | **Performance Efficiency** | パフォーマンス効率 |
    | **Cost Optimization** | コスト最適化 |
    | **Sustainability** | 持続可能性 |
