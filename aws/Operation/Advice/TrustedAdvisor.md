# AWS Trusted Advisor

## Overview
- AWS環境のベストプラクティスをチェックし、==推奨事項を提供==するサービス
  - チェック内容:
    - `コスト最適化`:       ==Business 以上== / 未使用リソース、リザーブド購入推奨
    - `パフォーマンス`:     ==Business 以上== / 過剰プロビジョニング、スループット
    - `セキュリティ`:       ==Basic 以上== / SG / IAM / MFA / S3 バケットのパブリックアクセス / EBS,RDS パブリックスナップショット
    - `耐障害性`:           ==Business 以上== / バックアップ、マルチAZ、冗長性
    - `サービス制限`:       ==Basic 以上== / クォータの使用率 (80%超で警告)
  - Organizations 統合:   委任管理者を設定して組織全体のチェック結果を集約 / 組織ビューで全アカウントの推奨事項を確認
