# Amazon Translate

==ニューラル機械翻訳==サービス。多言語テキストをリアルタイムで翻訳。

## 特徴
- `リアルタイム翻訳`:                  API で即時翻訳
- `バッチ翻訳`:                        S3 上の大量ドキュメントを一括翻訳
- `カスタム用語集`:                    専門用語・ブランド名の翻訳を指定
- `75+ 言語`:                          多言語対応

## SAA試験ポイント
- ==テキスト翻訳== → Translate
- ==多言語対応アプリ== → Translate
- ==音声をテキストに変換== → Transcribe (Translateではない)
