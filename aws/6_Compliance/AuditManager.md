# AWS Audit Manager

## Overview
- 監査自動化:                             ==監査準備==とコンプライアンス評価を==自動化== / エビデンス自動収集

---

## Artifact vs Audit Manager

| 項目 | ==Artifact== | ==Audit Manager== |
|------|-------------|-------------------|
| **用途** | AWS のレポート==ダウンロード== | ==自社の監査準備==を自動化 |
| **対象** | AWS のコンプライアンス | ==自社リソース==のコンプライアンス |
| **機能** | ドキュメント取得 | エビデンス収集・評価 |

---

## SAA 出題パターン

| シナリオ | 回答 |
|---------|------|
| 「==監査準備==を自動化したい」 | ==Audit Manager== |
| 「==コンプライアンス評価==のエビデンスを自動収集」 | ==Audit Manager== |
| 「AWS の SOC/PCI レポートをダウンロード」 | ==Artifact== (Audit Manager ではない) |
