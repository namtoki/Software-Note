# AWS License Manager

## Overview
- ライセンス管理:                         ソフトウェアライセンスの==使用状況を追跡==・==制限==を管理 / ==無料==

---

## 主な機能

| 機能 | 説明 |
|------|------|
| **使用状況追跡** | ライセンス消費量を可視化 |
| **ルール設定** | 使用上限を設定、超過防止 |
| **BYOL 管理** | ==Bring Your Own License== の追跡 |

---

## SAA 出題パターン

| シナリオ | 回答 |
|---------|------|
| 「ソフトウェア==ライセンスの使用状況==を管理」 | ==License Manager== |
| 「ライセンス==超過を防止==したい」 | ==License Manager== |
| 「==BYOL== (自社ライセンス) を AWS で追跡」 | ==License Manager== |
