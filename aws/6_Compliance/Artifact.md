# AWS Artifact

## Overview
- コンプライアンスレポート:               AWS の==セキュリティ・コンプライアンスドキュメント==をダウンロード / ==無料==
  - `Artifact Reports`:                 AWS のコンプライアンスレポート (SOC, PCI, ISO 等)
  - `Artifact Agreements`:              AWS との契約 (BAA, NDA 等) の確認・同意

---

## 取得可能なドキュメント例

| カテゴリ | 例 |
|---------|-----|
| **監査レポート** | SOC 1/2/3, PCI DSS |
| **認証** | ISO 27001, ISO 9001 |
| **契約** | BAA (HIPAA), NDA |
