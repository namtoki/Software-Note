# Amazon Comprehend

## Overview
- テキストからインサイトを抽出

## Options
- `感情分析`:                          ポジティブ / ネガティブ / ニュートラル / 混合
- `エンティティ抽出`:                  人名、場所、組織、日付等を識別
- `キーフレーズ抽出`:                  重要な語句を抽出
- `言語検出`:                          テキストの言語を判定
- `Comprehend Medical`:                ==医療用語== (病名、薬品名等) に特化
