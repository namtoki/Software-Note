# Amazon Rekognition

## Overview
- 画像・動画分析サービス:        機械学習による視覚的コンテンツ分析
  - 主要機能:
    - `顔検出・認識`:              顔の検出、比較、検索
    - `オブジェクト検出`:          物体、シーン、アクティビティの識別
    - `テキスト検出`:              画像内の文字を抽出 (OCR)
    - `コンテンツモデレーション`:  ==不適切コンテンツ==の検出
    - `有名人認識`:                著名人の識別
    - `PPE検出`:                   保護具 (ヘルメット等) の着用確認
  -  対応メディア:
    - `Image`:                     S3 または Base64 エンコード
    - `Video`:                     S3 / Kinesis Video Streams
