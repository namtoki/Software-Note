# AWS Proton

## 概要

**プラットフォームチーム**が標準化されたインフラテンプレートを作成し、**開発者**がセルフサービスでデプロイできるようにするサービス。

```
プラットフォームチーム          開発者
        ↓                        ↓
  テンプレート作成         テンプレート選択してデプロイ
  (インフラ標準化)         (インフラ知識不要)
```

## 解決する課題

| 課題 | Protonの解決策 |
|-----|---------------|
| 開発者ごとにインフラ構成がバラバラ | 標準テンプレートを提供 |
| 開発者がインフラに詳しくない | セルフサービスで簡単デプロイ |
| インフラ更新の反映が大変 | テンプレート更新を一括適用 |

## 主要コンポーネント

| コンポーネント | 説明 |
|--------------|------|
| **Environment Template** | 共有インフラ（VPC, Cluster等） |
| **Service Template** | アプリケーションインフラ（Lambda, ECS等） |
| **Environment** | テンプレートからデプロイされた環境 |
| **Service** | テンプレートからデプロイされたサービス |

## アーキテクチャ

```
┌─────────────────────────────────────────────────────┐
│  Environment Template (プラットフォームチーム作成)    │
│    - VPC                                            │
│    - ECS Cluster                                    │
│    - ALB                                            │
└─────────────────────────────────────────────────────┘
                    ↓ デプロイ
┌─────────────────────────────────────────────────────┐
│  Environment (dev / staging / prod)                 │
└─────────────────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────────────────┐
│  Service Template (プラットフォームチーム作成)        │
│    - ECS Service                                    │
│    - Task Definition                                │
│    - CI/CD Pipeline                                 │
└─────────────────────────────────────────────────────┘
                    ↓ 開発者がセルフサービスでデプロイ
┌─────────────────────────────────────────────────────┐
│  Service (アプリA / アプリB / アプリC)               │
└─────────────────────────────────────────────────────┘
```

## テンプレートエンジン

| エンジン | 説明 |
|---------|------|
| **CloudFormation** | AWS標準IaC |
| **Terraform** | HashiCorp製IaC |

## ワークフロー

```
1. プラットフォームチーム
   └── Environment / Service テンプレート作成・公開

2. 開発者
   └── カタログからテンプレート選択
   └── パラメータ入力（インスタンスサイズ等）
   └── デプロイ

3. テンプレート更新時
   └── プラットフォームチームが新バージョン公開
   └── 既存サービスに一括 or 段階的に適用
```

## 類似サービスとの比較

| サービス | 用途 |
|---------|------|
| **Proton** | 開発者向けセルフサービス + テンプレート管理 |
| **CloudFormation** | 汎用IaC |
| **Service Catalog** | IT製品のカタログ化 |
| **Elastic Beanstalk** | アプリデプロイ（PaaS） |

## ユースケース

| シナリオ | 説明 |
|---------|------|
| **マイクロサービス** | 各チームが標準構成でサービスをデプロイ |
| **ガバナンス** | セキュリティ・コンプライアンス準拠のテンプレート強制 |
| **オンボーディング** | 新チームがすぐにインフラを利用開始 |

## 試験ポイント

| シナリオ | 解答 |
|---------|------|
| 開発者がインフラを意識せずデプロイしたい | **AWS Proton** |
| プラットフォームチームがインフラを標準化 | **AWS Proton** |
| テンプレート更新を既存環境に一括適用 | **AWS Proton** |
| 汎用的なIaC | **CloudFormation** |
