# AWS Certificate Manager (ACM)

- `AWS Certificate Manager (ACM)`           SSL/TLS証明書を管理 / ==HTTPS通信== に必要な証明書を、無料で発行・更新・管理

## Overview
- SSL/TLS 証明書管理:                       ==パブリック証明書==を無料で発行 / ==自動更新==で運用負荷軽減

---

## 主な機能

| 機能 | 説明 |
|------|------|
| **証明書発行** | パブリック SSL/TLS 証明書を==無料==発行 |
| **自動更新** | 証明書の==有効期限を自動延長== |
| **インポート** | サードパーティ証明書も管理可能 |

---

## 統合サービス

| サービス | リージョン |
|---------|-----------|
| **ALB / NLB** | 各リージョン |
| **CloudFront** | ==us-east-1== のみ |
| **API Gateway** | 各リージョン |

---

## SAA 出題パターン

| シナリオ | 回答 |

| 「==SSL/TLS 証明書==を無料で管理したい」 | ==ACM== |
