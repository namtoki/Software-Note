# AWS Cost Explorer

```
過去のコストデータ
      ↓
分析・可視化・フィルタリング
      ↓
将来のコスト予測 (最大12ヶ月)
```

## Overview
- コスト分析サービス:                 コストと使用量を可視化・分析 / ==過去12ヶ月== / 月次、日次、==!時間単位 ($0.01/1,000リクエスト)==
  - `フィルタリング`:                   ==サービス、リージョン、タグ等で絞り込み==
  - `グループ化`:                       サービス別、アカウント別等で集計
- 予測:                               過去データから将来コストを==最大12ヶ月先== まで予測
- `RI / Savings Plans Recommendations`: 購入すべきRI/Savings Plansを提案
