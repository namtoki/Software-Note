# AWS Data Exchange

## Overview
- データマーケットプレイス:                 ==サードパーティのデータ==を検索・購読・利用 / S3 へ直接配信

---

## Marketplace vs Data Exchange

| 項目 | ==Marketplace== | ==Data Exchange== |
|------|-----------------|-------------------|
| **対象** | ==ソフトウェア== | ==データセット== |
| **用途** | アプリ購入・デプロイ | データ購読・分析 |
| **例** | AMI, SaaS | 金融データ, 気象データ等 |

---

## SAA 出題パターン

| シナリオ | 回答 |
|---------|------|
| 「==サードパーティのデータ==を購読したい」 | ==Data Exchange== |
| 「外部の==データセット==を S3 に取り込む」 | ==Data Exchange== |
| 「サードパーティのソフトウェアを購入」 | ==Marketplace== |
