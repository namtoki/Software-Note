# AWS Serverless Application Repository (SAR)

## Overview
- サーバーレスアプリのリポジトリ:           ==Lambda / SAM==ベースのアプリを公開・共有・デプロイ

---

## Marketplace vs SAR

| 項目 | ==Marketplace== | ==SAR== |
|------|-----------------|---------|
| **対象** | 汎用ソフトウェア | ==サーバーレスアプリ==のみ |
| **形式** | AMI / SaaS / コンテナ等 | ==Lambda / SAM== |
| **用途** | 商用ソフトウェア購入 | サーバーレスコード再利用 |

---

## SAA 出題パターン

| シナリオ | 回答 |
|---------|------|
| 「==サーバーレスアプリ==を共有・再利用したい」 | ==SAR== |
| 「==Lambda 関数==のテンプレートを公開」 | ==SAR== |
| 「サードパーティの商用ソフトウェアを購入」 | ==Marketplace== |
