# AWS Lake Formation

## Associate Level
- データレイク構築:                       ==S3 ベース==のデータレイクを簡単に構築・管理 / ==一元的なアクセス制御==

---

## 概念

```
┌─────────────────────────────────────────────────────┐
│                  Lake Formation                     │
│  ┌─────────────────────────────────────────────┐   │
│  │              データレイク (S3)               │   │
│  │  ┌─────────┐ ┌─────────┐ ┌─────────┐       │   │
│  │  │ DB データ│ │ ログ    │ │ ファイル │       │   │
│  │  └─────────┘ └─────────┘ └─────────┘       │   │
│  └─────────────────────────────────────────────┘   │
│       ↑ 取込み         ↓ クエリ                    │
│  RDS, DynamoDB    Athena, Redshift, EMR           │
└─────────────────────────────────────────────────────┘
```

---

## 主な機能

| 機能 | 説明 |
|------|------|
| **データ取込み** | RDS, S3, オンプレ等からデータレイクへ |
| ==アクセス制御== | ==列・行レベル==でのきめ細かい権限管理 |
| **データカタログ** | Glue Data Catalog と統合 |
| **暗号化** | 保存時・転送時の暗号化 |

---

## Lake Formation vs S3 直接管理

| 項目 | ==S3 直接== | ==Lake Formation== |
|------|-----------|-------------------|
| **権限管理** | バケットポリシー/IAM | ==一元管理== |
| **きめ細かい制御** | 困難 | ==列・行レベル== |
| **管理の複雑さ** | 高い | ==簡素化== |
