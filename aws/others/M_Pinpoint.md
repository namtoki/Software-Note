# Amazon Pinpoint

## Overview
- マーケティングコミュニケーションサービス:   顧客セグメントに対してマルチチャネルでメッセージ配信
  - `マルチチャネル`:                           Email, SMS, プッシュ通知, 音声
  - `セグメンテーション`:                       ユーザー属性・行動でターゲティング
  - `キャンペーン`:                             スケジュール配信, A/Bテスト
  - `分析`:                                     配信・開封・クリック率のトラッキング
