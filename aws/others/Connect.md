# Amazon Connect

==クラウドベースのコンタクトセンター==サービス。電話・チャット対応を簡単に構築。

## 特徴
- `オムニチャネル`:                    電話 (インバウンド/アウトバウンド), チャット, タスク
- `従量課金`:                          インフラ不要 / 使用した分だけ課金
- `コンタクトフロー`:                  ドラッグ&ドロップでIVR (自動音声応答) を設計

## 連携サービス
- `Lex`:                               ==チャットボット== / 自然言語理解
- `Lambda`:                            コンタクトフロー内でカスタム処理
- `S3`:                                通話録音の保存
- `Kinesis`:                           リアルタイムデータストリーミング

## SAA試験ポイント
- ==コンタクトセンター==が必要 → Amazon Connect
- ==チャットボット==統合 → Lex連携
- ==従量課金==でスケーラブル
