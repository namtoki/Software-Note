# Amazon SES (Simple Email Service)

==トランザクションメール==送信サービス。大量メールの送受信に対応。

## 特徴
- `送信`:                              SMTP / API (SDK) / コンソール
- `受信`:                              メール受信 → S3保存, Lambda実行, SNS通知
- `サンドボックス`:                    新規アカウントは==検証済みアドレスのみ==送信可 / 本番利用は申請必要

## ユースケース
- パスワードリセット、注文確認、通知メール
- マーケティングメール (大量配信)
- メール受信トリガーの自動処理

## 類似サービス比較 (SAA頻出)
| サービス | 用途 | 特徴 |
|---------|------|------|
| `SES` | ==トランザクションメール== | 大量送信, SMTP/API, 受信も可 |
| `Pinpoint` | ==マーケティング== | キャンペーン, セグメント, 分析 |
| `SNS` | ==Pub/Sub通知== | システム間通知, SMS, プッシュ |
