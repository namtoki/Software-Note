# AWS DataSync

```
オンプレミス                          AWS
+------------+                    +------------+
|  NFS/SMB   | ← DataSync Agent → |  S3/EFS    |
+------------+      (EC2 or VM)   +------------+
```

## Overview
- データ転送サービス:             (オンプレ|AWS) ↔ AWS 間の高速転送 / ==自動で暗号化・整合性検証・最適化== / 転送速度: ==最大10Gbps==
  - 対応ストレージ:
    - オンプレ:                   NFS, SMB, HDFS, オブジェクトストレージ
    - AWS:                        ==S3==, ==EFS==, ==FSx== (全種類)
  - `DataSync Agent`:               オンプレにデプロイ / ==AWS間転送では不要==
  - スケジュール実行:             ==定期的==な同期ジョブ
  - 帯域幅制限:                   業務時間中は制限、夜間はフル活用
  - 増分転送:                     ==変更分のみ転送== / 初回以降は高速
  - データ検証:                   転送中・転送後に整合性チェック

## DataSync vs 他サービス
| サービス | ユースケース |
|---------|-------------|
| `DataSync` | 大量データ移行・定期同期 (ネットワーク経由) |
| `Snowball` | 超大量データ (==ネットワーク帯域が不足時==) |
| `Transfer Family` | ==SFTP/FTP==でファイル転送 (既存ワークフロー維持) |
| `S3 Replication` | S3バケット間のオブジェクトレプリケーション |
