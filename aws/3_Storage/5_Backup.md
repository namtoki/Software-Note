## Overview
- 一元バックアップ管理:       複数 AWS サービスのバックアップを ==一元管理==
  - 対応サービス:             EC2, EBS, RDS, Aurora, DynamoDB, EFS, FSx, S3, Storage Gateway
  - `Backup Plan`:              スケジュール、保持期間、ライフサイクルを定義
  - `Backup Vault`:             バックアップデータの保存先 (==暗号化==)
    - `Vault Lock`:             用途: ==WORM== (Write Once Read Many) ポリシー / 効果: バックアップの ==削除・変更を防止== (コンプライアンス)
  - `クロスリージョン`:         別リージョンへバックアップコピー (==DR==)
  - `クロスアカウント`:         別アカウントへバックアップコピー (==セキュリティ==)
