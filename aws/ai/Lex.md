# Amazon Lex

==会話型AI== / ==チャットボット==構築サービス。Alexa と同じ技術。

## 特徴
- `音声 + テキスト`:                   両方に対応 / ASR (音声認識) + NLU (自然言語理解)
- `Alexa 同等`:                        Alexa と同じ深層学習技術
- `Lambda 統合`:                       バックエンドロジックを Lambda で実装

## 連携サービス
- `Amazon Connect`:                    ==コンタクトセンター==でチャットボット
- `Lambda`:                            ビジネスロジック実行
- `CloudWatch`:                        ログ・メトリクス

## SAA試験ポイント
- ==チャットボット== → Lex
- ==音声 + テキスト対応== → Lex
- ==Connect + チャットボット== → Connect + Lex 連携
