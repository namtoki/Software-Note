# Amazon Polly

==テキストを音声に変換== (Text-to-Speech) するサービス。

## 特徴
- `多言語`:                            60+ 言語、多数の音声
- `Neural TTS`:                        より自然な音声 (ニューラルネットワーク)
- `SSML`:                              発音、速度、ピッチ等をカスタマイズ
- `リアルタイム`:                      ストリーミング配信可能

## SAA試験ポイント
- ==テキスト → 音声== → Polly
- ==音声 → テキスト== → Transcribe (Pollyではない)
