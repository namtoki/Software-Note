# Amazon Forecast

AWS が提供する**機械学習ベースの時系列予測サービス**。

## 概要

- 過去のデータから将来の値を予測する
- 機械学習の専門知識がなくても利用可能
- Amazon.com で培った予測技術を利用

## ユースケース

| 分野 | 予測内容 |
|------|----------|
| 小売 | 商品の需要予測、在庫最適化 |
| EC | 売上予測 |
| 物流 | 配送量の予測 |
| 人事 | 必要人員の予測 |
| エネルギー | 電力需要の予測 |
| 財務 | 収益・キャッシュフロー予測 |

## 仕組み

```
過去のデータ（CSV/時系列）
        |
        v
   Amazon Forecast
   (自動で最適なアルゴリズム選択)
        |
        v
   予測モデル生成
        |
        v
   将来の値を予測
```

## 特徴

- **AutoML**: 複数のアルゴリズムを自動で試し、最適なものを選択
- **追加データ対応**: 天気、祝日、プロモーション情報などを加味した予測が可能
- **精度指標**: 予測の信頼区間も出力

## 料金

- 生成した予測数、トレーニングしたデータ量、保存データ量に基づく従量課金
