# Amazon Comprehend

==自然言語処理 (NLP)== サービス。テキストからインサイトを抽出。

## 機能
- `感情分析`:                          ポジティブ / ネガティブ / ニュートラル / 混合
- `エンティティ抽出`:                  人名、場所、組織、日付等を識別
- `キーフレーズ抽出`:                  重要な語句を抽出
- `言語検出`:                          テキストの言語を判定

## バリエーション
- `Comprehend`:                        汎用
- `Comprehend Medical`:                ==医療用語== (病名、薬品名等) に特化

## SAA試験ポイント
- ==テキスト感情分析== → Comprehend
- ==NLP / テキスト分析== → Comprehend
- ==医療テキスト分析== → Comprehend Medical
