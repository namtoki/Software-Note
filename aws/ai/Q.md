# Amazon Q

==生成AI搭載==のアシスタントサービス。

## タイプ
| タイプ | 用途 |
|-------|------|
| `Q Business` | ==企業データ==に基づく質問応答 / S3, SharePoint等と連携 |
| `Q Developer` | ==コーディング支援== / コード生成, デバッグ, 変換 (旧CodeWhisperer) |
| `Q in QuickSight` | ==BI分析== / 自然言語でダッシュボード作成 |
| `Q in Connect` | ==コンタクトセンター== / エージェント支援 |

## SAA試験ポイント
- ==生成AIアシスタント== → Amazon Q
- ==企業データ検索・質問応答== → Q Business
- ==コード生成・開発支援== → Q Developer
