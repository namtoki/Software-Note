# Amazon Neptune

## Associate Level
- グラフデータベース:                     ==関係性・つながり==を効率的に管理・クエリ / フルマネージド

---

## ユースケース (SAA 頻出)

| ユースケース | 説明 |
|-------------|------|
| ==ソーシャルネットワーク== | 友人関係、フォロー関係 |
| ==レコメンデーション== | 「この商品を買った人は...」 |
| ==不正検出== | 不正パターンの関係性分析 |
| ==ナレッジグラフ== | 情報間の関連付け |

---

## データベースの使い分け (SAA 頻出)

| 要件 | サービス |
|------|---------|
| リレーショナル (SQL) | ==RDS / Aurora== |
| キーバリュー / ドキュメント | ==DynamoDB== |
| ==グラフ (関係性)== | ==Neptune== |
| データウェアハウス (OLAP) | ==Redshift== |
| インメモリキャッシュ | ==ElastiCache== |
