# Amazon EMR (Elastic MapReduce)

```
+-----------------------------------------------------------+
|                       EMR Cluster                         |
|  +----------------+  +----------------+  +--------------+ |
|  |    Master      |  |     Core       |  |    Task      | |
|  | クラスター管理 |  | データ + 処理  |  |   処理のみ   | |
|  +----------------+  +----------------+  +--------------+ |
|           |                  |                  |         |
|           +------------------+------------------+         |
|                              |                            |
|  +----------------------------------------------------+   |
|  |     Hadoop / Spark / Presto / Hive / HBase         |   |
|  +----------------------------------------------------+   |
|                              |                            |
|  +----------------------------------------------------+   |
|  |              S3 (EMRFS) / HDFS                     |   |
|  +----------------------------------------------------+   |
+-----------------------------------------------------------+
```

## Overview
- ビッグデータ処理:       ==Hadoop / Spark== 等のマネージドクラスター / ==ペタバイト級==データ処理
  - 対応フレームワーク:
    | フレームワーク | 用途 |
    |---------------|------|
    | `Apache Spark` | 高速データ処理、ML |
    | `Apache Hadoop` | 分散処理 (MapReduce) |
    | `Presto / Trino` | インタラクティブクエリ |
    | `Hive` | SQL ライクなクエリ |
    | `HBase` | NoSQL データベース |
