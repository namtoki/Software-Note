# AWS Local Zones

## Overview
- 小規模データセンター:   Region を主要都市近くに拡張 / ==AWS 管理== / ==!Region より 10〜20% 高い== / ==マルチテナント== / 低レイテンシー

## ユースケース

| シナリオ | Local Zones が適切な理由 |
|---------|-------------------------|
| 都市圏の低レイテンシー | リージョンより近い場所で処理 |
| リアルタイムゲーム | プレイヤー近くにサーバー配置 |
| メディアレンダリング | 低レイテンシーが必要な処理 |
