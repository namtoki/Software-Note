# Amazon WorkSpaces

## Overview
- `WorkSpaces`:                            ==フルデスクトップ==を提供 / ==永続==ストレージでデータ保持
- `WorkSpaces Secure Browser`:             ==ブラウザのみ==を提供 / 社内 Web アプリへの安全なアクセス

---

## WorkSpaces ファミリー比較 (SAA 頻出)

| 項目 | ==WorkSpaces== | ==Secure Browser== | ==AppStream 2.0== |
|------|----------------|--------------------|--------------------|
| **配信対象** | ==デスクトップ全体== | ==ブラウザのみ== | ==アプリケーション== |
| **用途** | 仮想デスクトップ (VDI) | Web アプリアクセス | 特定アプリの配信 |
| **永続性** | ==永続== | ==非永続== | ==非永続== |
| **コスト** | 高 | ==低== | 中 |

---

## SAA 出題パターン

| シナリオ | 回答 |
|---------|------|
| 「==仮想デスクトップ== (VDI) が必要」 | ==WorkSpaces== |
| 「ユーザーに==フルデスクトップ==を提供」 | ==WorkSpaces== |
| 「リモートワーク用に==永続的==な環境を提供」 | ==WorkSpaces== |
| 「社内 ==Web アプリ==に安全にアクセスしたい」 | ==Secure Browser== |
| 「==ブラウザのみ==を低コストで提供」 | ==Secure Browser== |
| 「==特定アプリ==をブラウザで配信したい」 | ==AppStream 2.0== |
